Folder SoftwareX contains the version 1.0 described in Drachal (2023) https://doi.org/10.1016/j.softx.2023.101388
   
Folder IDUB contains the version 2.0 developed during the realization of the program "Excellence Initiative – Research University (IDUB)" through the project BOB-IDUB-622-89/2021 – Nowe Idee POB III IDUB, 01/IDUB/2019/94 at the University of Warsaw.

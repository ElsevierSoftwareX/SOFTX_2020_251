The purpose of creating this package was to allow to perform the dynamic mixture estimation with state-space components and normal regression components, and clustering with normal mixture. Quasi-Bayesian estimation, as well as, that based on the Kerridge inaccuracy approximation was implemented. Main references are: Nagy and Suzdaleva (2013) <doi:10.1016/j.apm.2013.05.038>; Nagy et al. (2011) <doi:10.1002/acs.1239>.  
For installation you need to download "dynmix_1.0.tar.gz" file and use install.packages(file_path,repos=NULL) where "file_path" is a path to the “dynmix_1.0.tar.gz” file on your machine. Alternatively, use install.packages("dynmix"). The package is also available on the CRAN R package repository: https://CRAN.R-project.org/package=dynmix  
For the use of the package, please, refer to the file "manual_v1.0.pdf".
